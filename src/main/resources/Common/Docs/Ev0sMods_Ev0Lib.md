**Short summary**

Ev0's OmniHoppers adds flexible item‑transport hoppers and a small management UI to Hytale. It extends container transfer behavior with advanced filters and a player-facing Wrench interaction to rotate hoppers in-world.

**Primary player-facing features**

- Universal Hopper Blocks: placeable hoppers that move items between inventories automatically.
- Filtering UI: open the hopper configuration UI to set allowed/blocked items or use simple whitelist/blacklist modes.
- Wrench Interaction: a small handheld tool (Wrench) lets operators and players rotate hoppers, or change modes in-world.
- Optional Upgrades: Tiers that move items at a multiplier that change hopper item transfer rate.
- Fluid Transfer - Place a hopper next to a fluid and watch it suck up a bucket of said fluid
- Arcio Compat - When ArcIO is installed, Hoppers can be turned off or on with ArcIO
[![q1EDlWu.md.png](https://iili.io/q1EDlWu.md.png)](https://freeimage.host/i/q1EDlWu)

**How players will use it in-game**

- Place the OmniHopper block next to chests, furnaces, crates, or other inventories.
- Right-click to open the configuration UI and set filters or select transfer mode.
- Items matching the enabled filters will be moved according to the hopper's current mode and available throughput.

**Server operator controls**
- Adjust global limits such as transfer rate and range in server config (if provided by your server pack). If no config is present, default values are used.

**Common scenarios / examples**

- Automated Sorting: chain OmniHoppers to route items from multiple input chests into a sorting area using filters.
- Bulk Transfer: connect storage rooms across distances by building hopper chains and scheduled transfer nodes.
- Processing Pipelines: combine OmniHoppers with machines to auto-feed furnaces and collect outputs.
- Fluid Transfer - Place a hopper next to a fluid and watch it suck up a bucket of said fluid
- Arcio Compat - When ArcIO is installed, Hoppers can be turned off or on with ArcIO

**Troubleshooting**

- Hopper not moving items: ensure adjacent inventories are accessible and not full; check filters are not blocking the intended items.


**Tips & best practices**

- Use filters to avoid accidental transfer of valuable items.
- Place a buffer chest between high-throughput sources to prevent congestion.
- Limit upgrade usage on competitive servers to preserve balance.
